# Ejercicio ListaCadenas

**Dada una lista de cadenas, eliminar las que tengan menos de 4 caracteres.**

Tengo un ArrayList de String llamado palabras. Meto algunas palabras aleatorias con diferentes longitudes de caracteres.
Despues, voy a utilizar un iterador para recorrer el ArrayList, comprobando si la longitud de una palabra es menos de 4 caracteres, lo elimina.
